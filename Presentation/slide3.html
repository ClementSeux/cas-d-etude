Le Processus de Recrutement

Parcours en 3 étapes

Sélection progressive, objective et rapide

Étape 1 1 semaine

Présélection CV

Responsable : RH

  • Compétences techniques clés
  • Expérience et projets réalisés
Output : 5 à 8 candidats
Étape 2 2 heures

Entretien Technique

Responsables : Lead Dev + CTO

  • Test pratique (1h)
  • Revue de code GitHub
Outils : CodePen / Repo
Étape 3 1 heure

Culture & Fit

Responsables : Culture Squad + Manager

  • Questions STAR
  • Valeurs et collaboration
Décision : Go / No-Go
